# Finance & Accounting Super-Skill

Comprehensive finance skill merging **Perplexity Computer's 7 finance + investment research skills** with **Claude Code's data science, data engineering, and financial analysis skills**.

## What's Inside

| Section | Description |
|---------|-------------|
| Financial Statements | GAAP templates, ratio analysis, DCF valuation |
| Journal Entries | 6 JE templates, month-end accrual checklist |
| Reconciliation | Bank recs, GL-to-subledger, reconciling items |
| Close Management | 15-day close calendar, task tracker |
| Variance Analysis | Price/volume/mix decomposition, waterfall narrative |
| Audit Support | SOX compliance, sample selection, workpapers |
| Markets & Investment | Real-time data, famous investor frameworks |
| Data Engineering | dbt models, Airflow DAGs, pipeline architecture |
| ML & Forecasting | Revenue forecasting, anomaly detection |
| Invoice Management | Parsing, categorization, retention |

## Sources Merged

**Perplexity Computer:** finance-markets, finance-financial-statements, finance-variance-analysis, finance-journal-entry-prep, finance-reconciliation, finance-close-management, finance-audit-support, investment-research

**Claude Code:** Invoice Organizer, Senior Data Scientist, Senior Data Engineer, Financial Analyst, Senior ML Engineer (alirezarezvani/claude-skills)
